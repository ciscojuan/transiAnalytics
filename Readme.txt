PROYECTO TRANSIANALYTICS PARA EL DIPLOMADO BIG DATA AND BUSSINESS INTELLIGENTS DE LA UNIVERSIDAD MINUTO DE DIOS 

CONTEXTO:
Este proyecto consiste en el desarrollo de un sistema de análisis web dirigido a organizaciones o instituciones
que trabajen con temas de tránsito y movilidad (Policía Nacional de Tránsito), este aplicativo permitirá a sus 
usuarios la visualización de reportes estadísticos acerca de los niveles de accidentalidad que se presenta en la ciudad de Bogotá.

OBJETIVO:
Desarrollo de un sitio web de reportes estadisticos y predictivos de accidentalidad en la ciudad de bogota.
